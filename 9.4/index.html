    Версия 9.4 (Март 2022)

    В данной статье приведено подробное описание всех изменений, включенных в релиз ГИС ГеоАналитика версии 9.4, включающий в себя модернизацию Модуля веерных запросов для проработки подачи сервисов по РРЛ и оптике, проработку скоростей свыше 100 Мбит/с, а также новый модуль для поиска потенциально переключаемых арендованных каналов.

    Модуль проработки веерных запросов

    Проверка включений OnNet

    Обновленная матрица

    В новой версии модуля веерных запросов реализована расширенная проверка возможности подачи сервиса OnNet с учетом ряда исключений в зависимости от типа оборудования.

    Проверка скоростей выше 100 Мбит/c

    Также в новой версии доступна возможность проверки возможности подачи сервиса со скоростями выше 100 Мбит/с в т.ч. по интерфейсам GE. При оценке возможности подачи сервиса не проводится проверка магистрального канала, поэтому результатом проверки всегда является "Условная ТВ", при этом в комментариях приводится дополнительная информация о необходимости проверки канала.

    Проверка включений по РРЛ

    В новой версии также реализована проверка возможности подачи сервиса по технологиям WLL/РРЛ: для этого последовательно с расширением радиуса от километра до ~15 (в зависимости от области) выполняется поиск эксплуатируемых базовых станций по данным системы NRI. Для каждой найденной точки выполняется анализ возможных вариантов подключения с учетом матрицы и подбирается оптимальный вариант и выполняется ориентировочный расчет стоимости подключения по аналогии с подключениями OnNet.
